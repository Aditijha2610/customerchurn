Customer Churn Analysis
Project Overview

Customer churn analysis is used to identify patterns and factors that cause customers to leave a service. This project analyzes a telecom customer churn dataset to understand customer behavior and identify key features that influence churn.

The goal of this project is to perform Exploratory Data Analysis (EDA) using Python to extract meaningful insights that can help businesses improve customer retention.

Dataset Description

The dataset contains customer information such as:

Customer ID

Gender

Senior Citizen status

Partner and Dependents

Tenure (number of months with the company)

Phone Service

Internet Service

Contract Type

Payment Method

Monthly Charges

Total Charges

Churn (Yes/No)

The target variable is Churn, which indicates whether a customer left the service.

Key Insights

Customers with month-to-month contracts have the highest churn rate.

Customers with long-term contracts (one year or two years) are less likely to churn.

Customers with dependents and partners tend to stay longer with the service.

New customers with low tenure are more likely to leave the service.

Senior citizens show slightly higher churn rates.

Conclusion

The analysis shows that contract type, tenure, and customer relationships play an important role in churn behavior. Businesses can reduce churn by encouraging long-term contracts and improving service for new customers.
